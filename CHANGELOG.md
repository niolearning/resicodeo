# Changelog

Todas las novedades importantes de resicodeo se documentan aquí.

## [1.0.0] — 2026-04-26

### 🎉 Lanzamiento inicial

#### Agregado
- Registro de cobros con cálculo automático de subtotal, IVA y retenciones
- Soporte para 4 tipos de cliente: persona física con/sin factura, persona moral, extranjero
- Soporte para actividad profesional vs empresarial
- Soporte para cotización con IVA aparte o incluido
- Registro de gastos con categorías para videógrafos
- Cálculo automático de IVA acreditable
- Resumen mensual con cálculo completo de ISR e IVA según tasas RESICO 2026
- Reportes anuales con gráfica de ingresos por mes
- Monitor del límite anual de RESICO ($3.5M)
- Saldos a favor automáticos
- Almacenamiento local entre sesiones
- Exportación de datos en JSON
- Diseño dark mode con identidad propia
- Tipografía Plus Jakarta Sans
- Sección de ayuda integrada con tasas y conceptos clave

### Tecnología
- HTML + CSS + JavaScript vanilla
- Single file (sin dependencias)
- localStorage para persistencia
- Compatible con todos los navegadores modernos
